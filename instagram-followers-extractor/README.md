Part of the 8bit Python series.

---

## 🚀 Installation

1. Clone the repo:

git clone https://github.com/Python741/instagram-followers-extractor.git

2. Enter the folder:

cd instagram-followers-extractor

3. Create virtual environment:

python -m venv venv

4. Activate:

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

5. Install dependencies:

pip install -r requirements.txt

6. Create your .env file:

Copy `.env.example` to `.env` and add your credentials.

7. Run:

python main.py
